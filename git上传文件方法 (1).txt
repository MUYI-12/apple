在命令行上创建一个新的存储库
git init 
git add 选票程序.c 
git commit -m“first commit” 
git remote add origin https://github.com/zhongxiaohang/web-.git
 git push -u origin master

命令的意思是把远程库中的更新合并到（pull=fetch+merge）本地库中，C-rebase的作用是取消掉本地库中刚刚的commit，并把他们接到更新后的版本库之中。
git pull --rebase origin master
